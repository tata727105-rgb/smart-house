This is an Arduino sketch that combines three main functions:

Distance detection (Ultrasonic sensor – HC-SR04)

Servo motor control

Temperature monitoring with LED alert

Hardware Used

Arduino Uno (or compatible)

Ultrasonic sensor (HC-SR04)

TRIG → pin 3

ECHO → pin 4

Servo motor → pin 9

Temperature sensor (LM35) → A0

LED → pin 2

Resistors, wires
